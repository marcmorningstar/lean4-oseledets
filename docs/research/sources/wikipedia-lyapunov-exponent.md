Rate of separation of infinitesimally close trajectories

[![](https://upload.wikimedia.org/wikipedia/commons/thumb/3/3f/Lyapunov-exponent.svg/500px-Lyapunov-exponent.svg.png)](https://en.wikipedia.org/wiki/File:Lyapunov-exponent.svg)

[![](https://upload.wikimedia.org/wikipedia/commons/thumb/3/3f/Orbital_instability_%28Lyapunov_exponent%29.png/500px-Orbital_instability_%28Lyapunov_exponent%29.png)](https://en.wikipedia.org/wiki/File:Orbital_instability_(Lyapunov_exponent).png)

Explanations of the Lyapunov exponent

In [mathematics](https://en.wikipedia.org/wiki/Mathematics "Mathematics")
, the **Lyapunov exponent** or **Lyapunov characteristic exponent** of a [dynamical system](https://en.wikipedia.org/wiki/Dynamical_system "Dynamical system")
 is a quantity that characterizes the exponential rate of separation of infinitesimally close [trajectories](https://en.wikipedia.org/wiki/Trajectory "Trajectory")
. Quantitatively, two trajectories in [phase space](https://en.wikipedia.org/wiki/Phase_space "Phase space")
 with initial separation vector δ 0 {\\displaystyle {\\boldsymbol {\\delta }}\_{0}} ![{\displaystyle {\boldsymbol {\delta }}_{0}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/3e54b6576f5802f07e3a30ba209ee68857b70dd2) diverge (provided that the divergence can be treated within the linearized approximation) at a rate given by

| δ ( t ) | ≈ e λ t | δ 0 | {\\displaystyle |{\\boldsymbol {\\delta }}(t)|\\approx e^{\\lambda t}|{\\boldsymbol {\\delta }}\_{0}|} ![{\displaystyle |{\boldsymbol {\delta }}(t)|\approx e^{\lambda t}|{\boldsymbol {\delta }}_{0}|}](https://wikimedia.org/api/rest_v1/media/math/render/svg/3f6d610ef7e59a7b4a2d1700aed68d5c66928b96) 

where λ {\\displaystyle \\lambda } ![{\displaystyle \lambda }](https://wikimedia.org/api/rest_v1/media/math/render/svg/b43d0ea3c9c025af1be9128e62a18fa74bedda2a) is the Lyapunov exponent.

The rate of separation can be different for different orientations of initial separation vector. Thus, there is a **spectrum of Lyapunov exponents**—equal in number to the dimensionality of the phase space. It is common to refer to the largest one as the **maximal Lyapunov exponent** (MLE), because it determines a notion of [predictability](https://en.wikipedia.org/wiki/Predictability "Predictability")
 for a dynamical system. A positive MLE is usually taken as an indication that the system is [chaotic](https://en.wikipedia.org/wiki/Chaos_theory "Chaos theory")
 (provided some other conditions are met, e.g., phase space compactness). This is because for the MLE to be more than 0, the average derivative needs to be more than 1, since we find its logarithms. If the average derivatives are above 1, it means that the function grows exponentially, which makes it so that any small change between initial values increase from their original value, so the function is divergent. If it is less than 0, it means that the average derivative is less than 1, which means that the function is decreasing, which would make any changes in the initial values decrease over time, making the function convergent. Note that an arbitrary initial separation vector will typically contain some component in the direction associated with the MLE, and because of the exponential growth rate, the effect of the other exponents will diminish over time.

The exponent is named after [Aleksandr Lyapunov](https://en.wikipedia.org/wiki/Aleksandr_Lyapunov "Aleksandr Lyapunov")
.

Definition of the Lyapunov spectrum
-----------------------------------

\[[edit](https://en.wikipedia.org/w/index.php?title=Lyapunov_exponent&action=edit&section=2 "Edit section: Definition of the Lyapunov spectrum")\
\]

[![Lyapunov exponent](https://upload.wikimedia.org/wikipedia/commons/thumb/0/04/LyapunovDiagram.svg/500px-LyapunovDiagram.svg.png)](https://en.wikipedia.org/wiki/File:LyapunovDiagram.svg)

The leading Lyapunov vector.

For a dynamical system with evolution equation x ˙ i \= f i ( x ) {\\displaystyle {\\dot {x}}\_{i}=f\_{i}(x)} ![{\displaystyle {\dot {x}}_{i}=f_{i}(x)}](https://wikimedia.org/api/rest_v1/media/math/render/svg/5b7f3cab0f75c97e6b4652afb5873eb6e6747ca9) in an _n_–dimensional phase space, the spectrum of Lyapunov exponents { λ 1 , λ 2 , … , λ n } , {\\displaystyle \\{\\lambda \_{1},\\lambda \_{2},\\ldots ,\\lambda \_{n}\\}\\,,} ![{\displaystyle \{\lambda _{1},\lambda _{2},\ldots ,\lambda _{n}\}\,,}](https://wikimedia.org/api/rest_v1/media/math/render/svg/b6d451c9bd8a3fa425e7a5357b23473ab6f19bfc) in general, depends on the starting point x 0 {\\displaystyle x\_{0}} ![{\displaystyle x_{0}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/86f21d0e31751534cd6584264ecf864a6aa792cf) . However, we will usually be interested in the [attractor](https://en.wikipedia.org/wiki/Attractor "Attractor")
 (or attractors) of a dynamical system, and there will normally be one set of exponents associated with each attractor. The choice of starting point may determine which attractor the system ends up on, if there is more than one. (For Hamiltonian systems, which do not have attractors, this is not a concern.) The Lyapunov exponents describe the behavior of vectors in the [tangent space](https://en.wikipedia.org/wiki/Tangent_space "Tangent space")

The matrix Y {\\displaystyle Y} ![{\displaystyle Y}](https://wikimedia.org/api/rest_v1/media/math/render/svg/961d67d6b454b4df2301ac571808a3538b3a6d3f) describes how a small change at the point x ( 0 ) {\\displaystyle x(0)} ![{\displaystyle x(0)}](https://wikimedia.org/api/rest_v1/media/math/render/svg/8f7176643e6d36fa7674dc79fdff1a4daa068f5d) propagates to the final point x ( t ) {\\displaystyle x(t)} ![{\displaystyle x(t)}](https://wikimedia.org/api/rest_v1/media/math/render/svg/d54c275db3a1e620737b58e143b0818107fa5f5c) . The limit Λ \= lim t → ∞ 1 2 t log ⁡ ( Y ( t ) Y T ( t ) ) {\\displaystyle \\Lambda =\\lim \_{t\\rightarrow \\infty }{\\frac {1}{2t}}\\log(Y(t)Y^{T}(t))} ![{\displaystyle \Lambda =\lim _{t\rightarrow \infty }{\frac {1}{2t}}\log(Y(t)Y^{T}(t))}](https://wikimedia.org/api/rest_v1/media/math/render/svg/18028f42f186b36e3185bf4f5aeae17ea3bfec84) defines a matrix Λ {\\displaystyle \\Lambda } ![{\displaystyle \Lambda }](https://wikimedia.org/api/rest_v1/media/math/render/svg/0ac0a4a98a414e3480335f9ba652d12571ec6733) (the conditions for the existence of the limit are given by the [Oseledets theorem](https://en.wikipedia.org/wiki/Oseledets_theorem "Oseledets theorem")
). The Lyapunov exponents λ i {\\displaystyle \\lambda \_{i}} ![{\displaystyle \lambda _{i}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/72fde940918edf84caf3d406cc7d31949166820f) are defined by the eigenvalues of Λ {\\displaystyle \\Lambda } ![{\displaystyle \Lambda }](https://wikimedia.org/api/rest_v1/media/math/render/svg/0ac0a4a98a414e3480335f9ba652d12571ec6733) .

The set of Lyapunov exponents will be the same for almost all starting points of an [ergodic](https://en.wikipedia.org/wiki/Dynamical_system#Ergodic_systems "Dynamical system")
 component of the dynamical system.

 λ m a x \= max j lim sup t → ∞ 1 t ln ⁡ α j ( X ( t ) ) . {\\displaystyle \\lambda \_{\\mathrm {max} }=\\max \\limits \_{j}\\limsup \_{t\\rightarrow \\infty }{\\frac {1}{t}}\\ln \\alpha \_{j}{\\big (}X(t){\\big )}.} ![{\displaystyle \lambda _{\mathrm {max} }=\max \limits _{j}\limsup _{t\rightarrow \infty }{\frac {1}{t}}\ln \alpha _{j}{\big (}X(t){\big )}.}](https://wikimedia.org/api/rest_v1/media/math/render/svg/f8900d575b62717ccaba52f175fef23fe86eb400) Lyapunov proved that if the system of the first approximation is regular (e.g., all systems with constant and periodic coefficients are regular) and its largest Lyapunov exponent is negative, then the solution of the original system is [asymptotically Lyapunov stable](https://en.wikipedia.org/wiki/Lyapunov_stability "Lyapunov stability")

### Perron effects of largest Lyapunov exponent sign inversion

\[[edit](https://en.wikipedia.org/w/index.php?title=Lyapunov_exponent&action=edit&section=4 "Edit section: Perron effects of largest Lyapunov exponent sign inversion")\
\]

In 1930 [O. Perron](https://en.wikipedia.org/wiki/Oskar_Perron "Oskar Perron")
 constructed an example of a second-order system, where the first approximation has negative Lyapunov exponents along a zero solution of the original system but, at the same time, this zero solution of the original nonlinear system is Lyapunov unstable. Furthermore, in a certain neighborhood of this zero solution almost all solutions of the original system have positive Lyapunov exponents. Also, it is possible to construct a reverse example in which the first approximation has positive Lyapunov exponents along a zero solution of the original system but, at the same time, this zero solution of original nonlinear system is Lyapunov stable.[\[3\]](https://en.wikipedia.org/wiki/Lyapunov_exponent#cite_note-2005-IEEE-Discrete-system-stability-Lyapunov-exponent-3)
[\[4\]](https://en.wikipedia.org/wiki/Lyapunov_exponent#cite_note-2007-IJBC-Lyapunov-exponent-4)
 The effect of sign inversion of Lyapunov exponents of solutions of the original system and the system of first approximation with the same initial data was subsequently called the Perron effect.[\[3\]](https://en.wikipedia.org/wiki/Lyapunov_exponent#cite_note-2005-IEEE-Discrete-system-stability-Lyapunov-exponent-3)
[\[4\]](https://en.wikipedia.org/wiki/Lyapunov_exponent#cite_note-2007-IJBC-Lyapunov-exponent-4)

Perron's counterexample shows that a negative largest Lyapunov exponent does not, in general, indicate stability, and that a positive largest Lyapunov exponent does not, in general, indicate chaos.

Therefore, time-varying linearization requires additional justification.[\[4\]](https://en.wikipedia.org/wiki/Lyapunov_exponent#cite_note-2007-IJBC-Lyapunov-exponent-4)

Basic properties
----------------

\[[edit](https://en.wikipedia.org/w/index.php?title=Lyapunov_exponent&action=edit&section=5 "Edit section: Basic properties")\
\]

If the system is conservative (i.e., there is no [dissipation](https://en.wikipedia.org/wiki/Dissipation "Dissipation")
), a [volume element](https://en.wikipedia.org/wiki/Volume_element "Volume element")
 of the phase space will stay the same along a trajectory. Thus the sum of all Lyapunov exponents must be zero. If the system is dissipative, the sum of Lyapunov exponents is negative.

If the system is a flow and the trajectory does not converge to a single point, one exponent is always zero—the Lyapunov exponent corresponding to the eigenvalue of L {\\displaystyle L} ![{\displaystyle L}](https://wikimedia.org/api/rest_v1/media/math/render/svg/103168b86f781fe6e9a4a87b8ea1cebe0ad4ede8) with an eigenvector in the direction of the flow.

Significance of the Lyapunov spectrum
-------------------------------------

\[[edit](https://en.wikipedia.org/w/index.php?title=Lyapunov_exponent&action=edit&section=6 "Edit section: Significance of the Lyapunov spectrum")\
\]

The Lyapunov spectrum can be used to give an estimate of the rate of [entropy production](https://en.wikipedia.org/wiki/Entropy_production "Entropy production")
, of the [fractal dimension](https://en.wikipedia.org/wiki/Fractal_dimension "Fractal dimension")
, and of the [Hausdorff dimension](https://en.wikipedia.org/wiki/Hausdorff_dimension "Hausdorff dimension")
 of the considered [dynamical system](https://en.wikipedia.org/wiki/Dynamical_system "Dynamical system")
.[\[5\]](https://en.wikipedia.org/wiki/Lyapunov_exponent#cite_note-2020-KuznetsovR-5)
 In particular from the knowledge of the Lyapunov spectrum it is possible to obtain the so-called [Lyapunov dimension](https://en.wikipedia.org/wiki/Lyapunov_dimension "Lyapunov dimension")
 (or [Kaplan–Yorke dimension](https://en.wikipedia.org/wiki/Kaplan%E2%80%93Yorke_conjecture "Kaplan–Yorke conjecture")
) D K Y {\\displaystyle D\_{KY}} ![{\displaystyle D_{KY}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/d99ac47e414853068c2be623f4a876c27a88765d) , which is defined as follows: D K Y \= k + ∑ i \= 1 k λ i | λ k + 1 | {\\displaystyle D\_{KY}=k+\\sum \_{i=1}^{k}{\\frac {\\lambda \_{i}}{|\\lambda \_{k+1}|}}} ![{\displaystyle D_{KY}=k+\sum _{i=1}^{k}{\frac {\lambda _{i}}{|\lambda _{k+1}|}}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/7d8c27c3a76dc60b78b7978c7419a729915614a4) where k {\\displaystyle k} ![{\displaystyle k}](https://wikimedia.org/api/rest_v1/media/math/render/svg/c3c9a2c7b599b37105512c5d570edc034056dd40) is the maximum integer such that the sum of the k {\\displaystyle k} ![{\displaystyle k}](https://wikimedia.org/api/rest_v1/media/math/render/svg/c3c9a2c7b599b37105512c5d570edc034056dd40) largest exponents is still non-negative. D K Y {\\displaystyle D\_{KY}} ![{\displaystyle D_{KY}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/d99ac47e414853068c2be623f4a876c27a88765d) represents an upper bound for the [information dimension](https://en.wikipedia.org/wiki/Information_dimension "Information dimension")
 of the system.[\[6\]](https://en.wikipedia.org/wiki/Lyapunov_exponent#cite_note-ky-6)
 Moreover, the sum of all the positive Lyapunov exponents gives an estimate of the [Kolmogorov–Sinai entropy](https://en.wikipedia.org/wiki/Kolmogorov%E2%80%93Sinai_entropy "Kolmogorov–Sinai entropy")
 accordingly to Pesin's theorem.[\[7\]](https://en.wikipedia.org/wiki/Lyapunov_exponent#cite_note-pesin-7)
 Along with widely used numerical methods for estimating and computing the [Lyapunov dimension](https://en.wikipedia.org/wiki/Lyapunov_dimension "Lyapunov dimension")
 there is an effective analytical approach, which is based on the direct Lyapunov method with special Lyapunov-like functions.[\[8\]](https://en.wikipedia.org/wiki/Lyapunov_exponent#cite_note-8)
 The Lyapunov exponents of bounded trajectory and the [Lyapunov dimension](https://en.wikipedia.org/wiki/Lyapunov_dimension "Lyapunov dimension")
 of attractor are invariant under [diffeomorphism](https://en.wikipedia.org/wiki/Diffeomorphism "Diffeomorphism")
 of the phase space.[\[9\]](https://en.wikipedia.org/wiki/Lyapunov_exponent#cite_note-9)

The [multiplicative inverse](https://en.wikipedia.org/wiki/Multiplicative_inverse "Multiplicative inverse")
 of the largest Lyapunov exponent is sometimes referred in literature as [Lyapunov time](https://en.wikipedia.org/wiki/Lyapunov_time "Lyapunov time")
, and defines the characteristic _e_\-folding time. For chaotic orbits, the Lyapunov time will be finite, whereas for regular orbits it will be infinite.

Numerical calculation
---------------------

\[[edit](https://en.wikipedia.org/w/index.php?title=Lyapunov_exponent&action=edit&section=7 "Edit section: Numerical calculation")\
\]

[![Lyapunov exponent](https://upload.wikimedia.org/wikipedia/commons/thumb/2/2d/Lyapunov_exponents_of_the_Mandelbrot_set_%28The_mini-Mandelbrot%29_-_Matlab.png/500px-Lyapunov_exponents_of_the_Mandelbrot_set_%28The_mini-Mandelbrot%29_-_Matlab.png)](https://en.wikipedia.org/wiki/File:Lyapunov_exponents_of_the_Mandelbrot_set_(The_mini-Mandelbrot)_-_Matlab.png)

Points inside and outside [Mandelbrot set](https://en.wikipedia.org/wiki/Mandelbrot_set "Mandelbrot set")
 colored by Lyapunov exponent.

Generally the calculation of Lyapunov exponents, as defined above, cannot be carried out analytically, and in most cases one must resort to numerical techniques. An early example, which also constituted the first demonstration of the exponential divergence of chaotic trajectories, was carried out by [R. H. Miller](https://en.wikipedia.org/w/index.php?title=Richard_H._Miller&action=edit&redlink=1 "Richard H. Miller (page does not exist)")
 in 1964.[\[10\]](https://en.wikipedia.org/wiki/Lyapunov_exponent#cite_note-10)
 Currently, the most commonly used numerical procedure estimates the L {\\displaystyle L} ![{\displaystyle L}](https://wikimedia.org/api/rest_v1/media/math/render/svg/103168b86f781fe6e9a4a87b8ea1cebe0ad4ede8) matrix based on averaging several finite time approximations of the limit defining L {\\displaystyle L} ![{\displaystyle L}](https://wikimedia.org/api/rest_v1/media/math/render/svg/103168b86f781fe6e9a4a87b8ea1cebe0ad4ede8) .

One of the most used and effective numerical techniques to calculate the Lyapunov spectrum for a smooth dynamical system relies on periodic [Gram–Schmidt](https://en.wikipedia.org/wiki/Gram%E2%80%93Schmidt "Gram–Schmidt")
 orthonormalization of the [Lyapunov vectors](https://en.wikipedia.org/wiki/Lyapunov_vector "Lyapunov vector")
 to avoid a misalignment of all the vectors along the direction of maximal expansion.[\[11\]](https://en.wikipedia.org/wiki/Lyapunov_exponent#cite_note-benettin-11)
[\[12\]](https://en.wikipedia.org/wiki/Lyapunov_exponent#cite_note-12)
[\[13\]](https://en.wikipedia.org/wiki/Lyapunov_exponent#cite_note-shimada-13)
[\[14\]](https://en.wikipedia.org/wiki/Lyapunov_exponent#cite_note-14)
 The Lyapunov spectrum of various models are described.\[_[clarification needed](https://en.wikipedia.org/wiki/Wikipedia:Please_clarify "Wikipedia:Please clarify")\
_\][\[15\]](https://en.wikipedia.org/wiki/Lyapunov_exponent#cite_note-15)
 Source code is provided \[_[clarification needed](https://en.wikipedia.org/wiki/Wikipedia:Please_clarify "Wikipedia:Please clarify")\
_\] for several [nonlinear systems](https://en.wikipedia.org/wiki/Nonlinear_system "Nonlinear system")
, including the [Hénon map](https://en.wikipedia.org/wiki/H%C3%A9non_map "Hénon map")
, the [Lorenz system](https://en.wikipedia.org/wiki/Lorenz_system "Lorenz system")
, and a [delay differential equation](https://en.wikipedia.org/wiki/Delay_differential_equation "Delay differential equation")
.[\[16\]](https://en.wikipedia.org/wiki/Lyapunov_exponent#cite_note-16)
[\[17\]](https://en.wikipedia.org/wiki/Lyapunov_exponent#cite_note-17)
[\[18\]](https://en.wikipedia.org/wiki/Lyapunov_exponent#cite_note-18)

Calculating Lyapunov exponents from limited experimental data presents significant challenges. The primary difficulty arises because such data is typically confined to the system's attractor and does not explore the full [phase space](https://en.wikipedia.org/wiki/Phase_space "Phase space")
. This confinement results in a 'singular' dataset with minimal extension along certain directions. These singular directions are precisely those associated with the more negative Lyapunov exponents, making them notoriously difficult to determine accurately.[\[19\]](https://en.wikipedia.org/wiki/Lyapunov_exponent#cite_note-bryant2-19)

To address this, one effective technique is to use nonlinear mappings to model the evolution of small displacements from the attractor. This approach can dramatically improve the ability to recover the full Lyapunov spectrum,[\[20\]](https://en.wikipedia.org/wiki/Lyapunov_exponent#cite_note-Bryant1-20)
[\[21\]](https://en.wikipedia.org/wiki/Lyapunov_exponent#cite_note-brown-21)
 but it is highly sensitive to noise and requires very clean data to be effective.

Local Lyapunov exponent
-----------------------

\[[edit](https://en.wikipedia.org/w/index.php?title=Lyapunov_exponent&action=edit&section=8 "Edit section: Local Lyapunov exponent")\
\]

Whereas the (global) Lyapunov exponent gives a measure for the total predictability of a system, it is sometimes of interest to estimate the local predictability around a point _x_0 in phase space. This may be done through the [eigenvalues](https://en.wikipedia.org/wiki/Eigenvalues "Eigenvalues")
 of the [Jacobian](https://en.wikipedia.org/wiki/Jacobian_matrix_and_determinant "Jacobian matrix and determinant")
 matrix _J_0(_x_0). These eigenvalues are also called local Lyapunov exponents.[\[22\]](https://en.wikipedia.org/wiki/Lyapunov_exponent#cite_note-22)
 Local exponents are not invariant under a nonlinear change of coordinates.

Conditional Lyapunov exponent
-----------------------------

\[[edit](https://en.wikipedia.org/w/index.php?title=Lyapunov_exponent&action=edit&section=9 "Edit section: Conditional Lyapunov exponent")\
\]

This term is normally used regarding [synchronization of chaos](https://en.wikipedia.org/wiki/Synchronization_of_chaos "Synchronization of chaos")
, in which there are two systems that are coupled, usually in a unidirectional manner so that there is a drive (or master) system and a response (or slave) system. The conditional exponents are those of the response system with the drive system treated as simply the source of a (chaotic) drive signal. Synchronization occurs when all of the conditional exponents are negative.[\[23\]](https://en.wikipedia.org/wiki/Lyapunov_exponent#cite_note-23)

### Software

\[[edit](https://en.wikipedia.org/w/index.php?title=Lyapunov_exponent&action=edit&section=13 "Edit section: Software")\
\]

*   [\[1\]](https://web.archive.org/web/20161027044059/http://www.mpipks-dresden.mpg.de/~tisean/Tisean_3.0.1/index.html)
     R. Hegger, H. Kantz, and T. Schreiber, Nonlinear Time Series Analysis, [TISEAN](https://en.wikipedia.org/wiki/Tisean "Tisean")
     3.0.1 (March 2007).
*   [\[2\]](https://web.archive.org/web/20130917012451/http://www.scientio.com/Products/ChaosKit)
     Scientio's ChaosKit product calculates Lyapunov exponents amongst other Chaotic measures. Access is provided online via a web service and Silverlight demo.
*   [\[3\]](http://ftp2.sco.com/pub/skunkware/src/x11/misc/mathrec-1.1c.tar.gz)
     [Archived](https://web.archive.org/web/20220628200022/http://ftp2.sco.com/pub/skunkware/src/x11/misc/mathrec-1.1c.tar.gz)
     2022-06-28 at the [Wayback Machine](https://en.wikipedia.org/wiki/Wayback_Machine "Wayback Machine")
     Dr. Ronald Joe Record's mathematical recreations software laboratory includes an X11 graphical client, lyap, for graphically exploring the Lyapunov exponents of a forced [logistic map](https://en.wikipedia.org/wiki/Logistic_map "Logistic map")
     and other maps of the unit interval. The [contents and manual pages](http://ftp2.sco.com/pub/skunkware/src/x11/misc/mathrec-1.1c/ReadMe.html)
     of the mathrec software laboratory are also available.
*   [\[4\]](http://biocircuits.ucsd.edu/pbryant/)
     This software provides a pair of powerful tools for calculating the full spectrum of Lyapunov exponents. The suite includes `LyapOde` for systems with known equations of motion and `Lyap` for analyzing experimental time series data. `LyapOde` ([C](https://en.wikipedia.org/wiki/C_(programming_language) "C (programming language)")
     source code included) is highly flexible, allowing users to supply their own model equations without limits on variables, and can also compute conditional Lyapunov exponents for coupled systems. Alternatively, `Lyap` ([Fortran](https://en.wikipedia.org/wiki/Fortran "Fortran")
     source code included) can determine Lyapunov direction vectors and characterize attractor singularity, a key feature for analyzing time series. Both tools are [cross-platform](https://en.wikipedia.org/wiki/Cross-platform_software "Cross-platform software")
     ([Windows](https://en.wikipedia.org/wiki/Windows "Windows")
    , [macOS](https://en.wikipedia.org/wiki/MacOS "MacOS")
    , [Linux](https://en.wikipedia.org/wiki/Linux "Linux")
    /[Unix](https://en.wikipedia.org/wiki/Unix "Unix")
    ), come with extensive documentation and sample files, and run in a text-based environment. While they lack built-in graphics, they generate output files ready for visualization in standard plotting software, such as [Microsoft Excel](https://en.wikipedia.org/wiki/Microsoft_Excel "Microsoft Excel")
    .
